# Modul 01: Landing Page

## Syfte
Fånga leads (email + preferenser) från potentiella kunder.

## Funktioner
- Professionell landing page på svenska
- Formulär: email, företag, telefon, bransch-val, region
- Success-meddelande efter registrering
- Mobile-responsive design

## Input
- Användarens formulärdata (email, företag, bransch, region)

## Output
- Lead data sparas (för MVP: localStorage + Google Sheets senare)

## Framtida integration
När Onboarding-modulen är klar:
- Skicka lead till databasen
- Initiera välkomstmejl

## Test
Öppna `index.html` i webbläsare och testa:
1. Formulär-submit → success-meddelande visas
2. Validering fungerar (required fields)
3. Checkbox-grupperna funkar
4. Mobile-läge ser ok ut

- `index.html` - Huvudsida
